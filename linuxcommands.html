linux commands

  • mkdir filename

    • mkdir is used to make a directory

  • rmdir filename

    • rmdir is used to remove a directory

  • cat>filename

    • cat command is used to create a file

  • cat filename

    • cat command is used to edit a file

  • cp filename1 filename2 ~/ location

    • cat command is used to copy a file

  • mv filename1 filename2 ~/ location

    • mv command is used to rename and move a file

  • ls

    • ls command is used to list all files and folders

  • ls -l

    • ls command is used to long list all files and folders

  • rm

    • rm is used to remove a file

  • sudo nautilus

    • run file manager as superuser

  • chmod 666 filename

    • allow read and write permissions to file
